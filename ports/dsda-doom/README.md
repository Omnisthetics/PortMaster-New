## Controls

| Button | Action |
|--|--| 
|Left Stick|Movement|
|Right Stick|Aim|
|R1|Fire|
|R2|Use Item|
|L1|Map|
|A|Interact|
|X|Previous Weapon|
|Y|Next Weapon|
|Left DPAD|Previous Item|
|Right DPAD|Next Item|
|R3|180-Degree Turn|
|L3|Toggle Autorun|
|Start|Menu|
